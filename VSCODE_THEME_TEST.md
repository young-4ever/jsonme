## 🐛 常见问题排查

### 问题1: 行号不显示
**解决方法**: 
- 检查 `jsonLines` 计算属性是否正确
- 确认 Vue 的 v-for 循环正常工作

### 问题2: 语法高亮不生效
**解决方法**:
- 检查 CSS 样式是否正确加载
- 确认正则表达式匹配正常
- 验证 v-html 渲染正确

### 问题3: 交通灯不显示
**解决方法**:
- 检查 CSS 的 border-radius 和 background 属性
- 确认 flexbox 布局正常

### 问题4: 响应式不工作
**解决方法**:
- 检查媒体查询是否正确
- 确认 viewport meta 标签存在
